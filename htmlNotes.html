Introduction to Html

What is HTML?

HTML (HyperText Markup Language)     :-     is the standard markup language used to create and structure pages on the web. It describes the content and layout of a webpage using a system of tags and attributes, allowing browsers to display text, images, links, and other elements correctly.

Why you use HTML?

We use HTML because it is the fundamental language for:

  • Structuring web content – defining headings, paragraphs, lists, tables, and forms.
  • Embedding media – adding images, audio, and videos to web pages.
  • Creating hyperlinks – linking pages and resources together to form the World Wide Web.
  • Ensuring browser compatibility – it’s supported by all web browsers, making it the universal standard for web content.

What are the features of HTML?

  • All instructions should pass through tags.
  • HTML tags are predefined.
  • Not case sensitive.

Basic HTML Text Formatting Tags

  • <b> – This tag makes the text bold. It’s used just for styling and doesn’t give any extra importance to the content.
    • Example: This is bold → This is bold
  • <strong> – This also makes the text bold, but it adds importance or seriousness to the content.
    • Example: Important! → Important!
  • <i> – This tag makes the text italic. Like <b>, it is just for styling.
    • Example: This is italic → This is italic
  • <em> – This makes the text italic, but it also adds emphasis or stress on the word.
    • Example: Don't do that! → Don't do that!
  • <u> – This tag underlines the text.
    • Example: This is underlined → This is underlined
  • <small> – This tag makes the text appear smaller than the normal size.
    • Example: This is small text → This is small text
  • <del> – This is used to show deleted text with a line through it (strikethrough).
    • Example: Old Price → Old Price
  • <strike> – This tag was used to show text with a strikethrough (a line through it). It's now replaced by <del> in HTML5.
    • Example: Old content → Old content
  • <sub> – This tag is for subscript text (appears below the line), often used in scientific formulas.
    • Example: H2O → H₂O
  • <sup> – This tag is for superscript text (appears above the line), like exponents.
    • Example: x2 → x²

HTML Media and Link Tags

  • <img> – Used to embed an image.
    • Common attributes:
      • src – Source of the image
      • alt – Alternate text if image fails
      • width – Width of the image
      • height – Height of the image
      • title – Tooltip on hover
    • Example: <img src="image.jpg" alt="Description">
  • <a> </a> – Used to create a hyperlink.
    • Common attributes:
      • href – URL of the link
      • target – Where to open the link (_blank, _self)
      • title – Tooltip on hover
      • download – Download the linked resource
    • Example: <a href="https://example.com">Visit Site</a>

HTML List Tags

  • Ordered list(<ol> </ol> )
    • Common attributes:
      • type – Type of numbering (e.g., 1, A, a, I, i)
      • start – Starting number (e.g., start="5")
      • reversed – Reverse the numbering (e.g., reversed)
    • Example:
      <ol type="A" start="3">
        <li>First item</li>
        <li>Second item</li>
      </ol>
  • Unordered list (<ul> </ul>)
    • Common attributes: *(No major attributes in modern HTML)*
    • Note: Earlier type attribute was used (like circle, square), but it's deprecated. Use CSS instead.
    • Example:
      <ul>
        <li>Item 1</li>
        <li>Item 2</li>
      </ul>
  • <li> </li> – List item inside <ul> or <ol>.
    • Common attributes: *(Usually styled using CSS)*
  • description list (<dl> </dl>) – This tag is used to define a Description List. It contains terms and their descriptions using the <dt> (term) and <dd> (description) tags.
    Example:
    <dl>
      <dt>HTML</dt>
      <dd>HyperText Markup Language</dd>
      <dt>CSS</dt>
      <dd>Cascading Style Sheets</dd>
    </dl>
    • Table tag (<table> </table>) – This tag is used to define an HTML Table. It arranges data in rows and columns using other related tags like <tr> (table row), <th> (table heading), and <td> (table data).
    Example:
    <table border="1">
      <tr>
        <th>Name</th>
        <th>Age</th>
      </tr>
      <tr>
        <td>Kiran</td>
        <td>22</td>
      </tr>
      <tr>
        <td>hemanth</td>
        <td>23</td>
      </tr>
    </table>
  • Semantic Tags: Tags that clearly describe their meaning to both the browser and developer.
    • <header> </header> – Page or section header
    • <nav> </nav> – Navigation links
    • <main> </main> – Main content of the page
    • <section> </section> – A section in a document
    • <footer> </footer> – Page or section footer
  • Block-Level Elements: These elements start on a new line and take up full width.
    • <div> </div> – Generic block container
    • <p> </p> – Paragraph
    • <h1> to <h6> – Headings
    • <ul>, <ol> – Unordered/Ordered lists
    • <table> – Table
  • Inline Elements: These elements do not start on a new line and take up only as much width as needed.
    • <span> </span> – Generic inline container
    • <a> </a> – Anchor (hyperlink)
    • <strong> </strong> – Important text (bold)
    • <em> </em> – Emphasized text (italic)
    • <img> – Image
  • <audio> Tag: Used to embed sound/audio content.
    • Attributes: src, controls, autoplay, loop
    • Example: <audio controls src="sound.mp3"></audio>
  • <video> Tag: Used to embed video content.
    • Attributes: src, controls, autoplay, loop, poster
    • Example: <video controls src="video.mp4"></video>
  • <form> Tag: Used to create an HTML form for user input.
  • <input> Tag: Represents an input field where the user can enter data.
    • Attributes: type, name, value, placeholder, size, maxlength, required
    • Example: <input type="text" name="username" placeholder="Enter name">
  • <label> Tag: Defines a label for an <input> element.
    • Attribute: for (must match the input's id)
    • Example: <label for="email">Email:</label>
  • type (Attribute): Specifies the type of <input>. Determines what kind of data the input accepts.
    • Examples: text, password, email, number, checkbox, radio, submit, file
    • Example: <input type="email" name="email">
  • for (Attribute): Used in the <label> tag to bind the label to an input element.
    • Example: <label for="user">Username</label> with <input id="user">
  • <meta> Tag: Provides metadata about the HTML document (not visible to users).
    • Common Uses: charset, viewport, description, keywords, author
    • Example: <meta charset="UTF-8">
  • size (Attribute): Specifies the **visible width** of an input field (in characters).
    • Example: <input type="text" size="30">
  • maxlength (Attribute): Limits the **maximum number of characters** allowed in an input field.
    • Example: <input type="text" maxlength="10">
  • required (Attribute): Specifies that the input **must be filled out** before submitting the form.
    • Example: <input type="email" required>
  • Radio Button: Allows the user to select one option from a set.
    • Example: <input type="radio" name="gender" value="male"> Male
  • Checkbox: Allows the user to select multiple options from a set.
    • Example: <input type="checkbox" name="hobby" value="music"> Music
  • File Field: Allows the user to upload a file from their device.
    • Example: <input type="file" name="resume">
  • Range: Allows the user to select a numeric value within a range using a slider.
    • Example: <input type="range" min="0" max="100">
  • Drop-down List (Select Box): Allows the user to select one option from a list.
    • Example: <select name="country">
        <option value="india">India</option>
        <option value="usa">USA</option>
      </select>
  • Multi-line Text (Textarea): Allows the user to enter long text or multiple lines.
    • Example: <textarea name="comments">Enter your comments here...</textarea>
  • Button: A clickable button used to submit, reset, or perform custom actions.
    • Example: <button type="submit">Submit</button>
  • Fieldset and Legend: Used to group related form elements and give a caption.
    • Example:
      <fieldset>
        <legend>Personal Info</legend>
        <input type="text" name="name">
      </fieldset>
  • iframe (Inline Frame): Used to embed another HTML page or external content within the current page.
    • Example:
      <iframe src="https://www.example.com" width="300" height="200"></iframe>
  • pattern (Attribute): Used in <input> to define a regular expression that the input's value must match.
    • Example: Only allow 5-digit numbers:
      <input type="text" pattern="[0-9]{5}" title="Enter 5 digits">
    • Note: The title attribute provides a message when the pattern doesn't match.

    • CSS(Cascading style Sheets)

  • CSS (Cascading Style Sheets): Used to style and design HTML elements (like colors, layouts, fonts, etc.).
    • Purpose: Enhances the visual appearance and layout of webpages.
  • Types of CSS Integration: CSS can be added to HTML in 3 main ways:
    • 1. Inline CSS: CSS written inside an element using the style attribute.
      • Example:
        <p style="color: red;">This is red text.</p>
    • 2. Internal CSS: CSS written inside a <style> tag in the <head> section.
      • Example:
        <style>
          p { color: green; }
        </style>
    • 3. External CSS: CSS written in a separate .css file and linked to the HTML using the <link> tag.
      • Example:
        <link rel="stylesheet" href="styles.css">
  • CSS Selectors: Used to select the HTML elements you want to style.
    • 1. Universal Selector (*) – Selects all elements.
      • Example: * { margin: 0; padding: 0; }
    • 2. Element Selector – Selects elements by tag name.
      • Example: p { color: blue; }
    • 3. ID Selector (#) – Selects an element by ID.
      • Example: #main { font-size: 18px; }
    • 4. Class Selector (.) – Selects elements by class.
      • Example: .box { border: 1px solid black; }
    • 5. Grouping Selector – Applies the same style to multiple elements.
      • Example: h1, h2, h3 { color: gray; }
    • 6. Pseudo-class Selectors – Apply styles to elements based on their state or position in the document.
      • Definition: Pseudo-classes are keywords added to selectors that define a special state of the selected elements.
      • Examples:
        • :hover – When the mouse pointer is over an element
          Example: a:hover { color: red; }
        • :focus – When an element (like input) is focused
          Example: input:focus { border: 2px solid blue; }
        • :first-child – Selects the first child of a parent
          Example: p:first-child { font-weight: bold; }
        • :last-child – Selects the last child
          Example: li:last-child { color: green; }
        • :nth-child(n) – Selects the nth child
          Example: li:nth-child(2) { background: yellow; }
      • Link-related Pseudo-classes:
        • :link – Normal, unvisited link
          Example: a:link { color: blue; }
        • :visited – Link that has been visited
          Example: a:visited { color: purple; }
        • :hover – When the mouse is over the link
          Example: a:hover { color: red; }
        • :active – When the link is being clicked
          Example: a:active { color: green; }
      • Tip: Always follow the order for links – :link → :visited → :hover → :active (L-V-H-A).

    CSS Properties

    • Color and Background Properties:
      • color – Sets the color of text
        Ways to specify values:
        • Named Colors → color: red;
        • Hex Codes → color: #ff0000;
        • RGB → color: rgb(255, 0, 0);
        • RGBA (with opacity) → color: rgba(255, 0, 0, 0.5);
        • HSL → color: hsl(0, 100%, 50%);
        • HSLA → color: hsla(0, 100%, 50%, 0.5);
      • background – Shorthand for background styling
        Properties we can include:
        • Color → background-color: yellow;
        • Image → background-image: url("image.jpg");
        • Repeat → background-repeat: no-repeat;
        • Position → background-position: center;
        • Size → background-size: cover;
        • Attachment → background-attachment: fixed;
        Example:
        background: url("bg.jpg") no-repeat center/cover fixed;
    • CSS Box Model:
      • Content: Actual content like text or image.
      • Padding: Space between content and border.
      • Border: Surrounds the padding and content.
      • Margin: Space outside the border (separates from other elements).
    • Border Properties:
      • border-width → border-width: 2px;
      • border-style → border-style: solid; (dashed, dotted, etc.)
      • border-color → border-color: blue;
      • Shorthand: border: 2px solid blue;
    • Outline:
      • Definition: An outline is a line drawn outside the element’s border. It is used mainly for highlighting elements (like focus or error states).
      • Unlike borders, outlines do not take up space and do not affect layout.
      • It cannot be applied to individual sides (no outline-left or outline-top).
    • Outline Properties:
      • outline-width → outline-width: 2px;
      • outline-style → outline-style: dotted;
      • outline-color → outline-color: red;
      • Shorthand: outline: 2px dotted red;
    • Outline Offset:
      • outline-offset – Moves the outline away from the border without affecting layout.
      • Example: outline-offset: 5px; → outline appears 5px outside the border.
    • Comparison: Margin vs Outline-Offset
      • margin adds space around the whole element, and it affects layout.
      • outline-offset creates space between the border and the outline only, without affecting layout.
      • They are not interchangeable.
      Example:
      .btn1 { border: 2px solid black; outline: 3px solid red; outline-offset: 5px; }
      .btn2 { border: 2px solid black; outline: 3px solid red; margin: 5px; }
      Explanation: The outline in .btn1 appears 5px away from the border. In .btn2, the element moves, but the outline still sticks to the border.
  • Padding Properties:
    • padding-top → padding-top: 10px;
    • padding-right → padding-right: 15px;
    • padding-bottom → padding-bottom: 10px;
    • padding-left → padding-left: 15px;
    • Shorthand: padding: 10px 15px;
      Format: padding: top right bottom left;
      Example: padding: 10px 20px 15px 5px;
  • Margin Properties:
    • margin-top → margin-top: 10px;
    • margin-right → margin-right: 15px;
    • margin-bottom → margin-bottom: 10px;
    • margin-left → margin-left: 15px;
    • Shorthand: margin: 10px 15px;
      Format: margin: top right bottom left;
      Example: margin: 5px auto; (for centering)
    • Special Values: margin: auto; (Used to center elements horizontally)
  • Display Property:
    • display: block; – Element takes full width, starts on new line
    • display: inline; – Element takes only the needed width, does not start on a new line
    • display: inline-block; – Like inline but can set width/height
    • display: none; – Hides the element completely (removes from layout)
    • display: flex; – Makes container a flexbox
    • display: grid; – Enables grid layout
    • Example: display: flex;
  • Position Property:
    • position: static; – Default value. The element stays in the normal document flow.
      Note: You can't use top/right/bottom/left with static.
    • position: relative; – Moves the element relative to its normal position (without affecting others).
      Use Case: Useful for fine adjustments or when you want to absolutely position children.
      Example: position: relative; top: 10px;
    • position: absolute; – Positions the element relative to the nearest positioned ancestor (anything except static).
      Important: If no such ancestor exists, it's relative to the <html> element (page).
      Use Case: Tooltips, dropdowns, overlays.
    • position: fixed; – Positions the element relative to the browser window (viewport).
      Stays fixed even when scrolling.
      Use Case: Sticky headers, floating action buttons.
    • position: sticky; – Acts like relative until a scroll threshold is reached, then becomes fixed.
      Use Case: Section headers that stay on top until a new section begins.
      Example: position: sticky; top: 0;
    • Common Offsets:
      • top, right, bottom, left – Used to position the element.
      • z-index – Controls the stack order (which element appears on top).
    • Example: position: absolute; top: 50px; left: 100px; z-index: 2;
  • Font Properties:
    • font-family – Specifies the typeface.
      Example: font-family: Arial, sans-serif;
    • font-size – Controls the size of the text.
      Example: font-size: 16px;
    • font-weight – Controls the thickness/boldness of text.
      Values: normal, bold, lighter, bolder, or 100–900
      Example: font-weight: 600;
    • font-style – Sets the font to italic or oblique.
      Values: normal, italic, oblique
      Example: font-style: italic;
    • font-variant – Displays small-caps.
      Example: font-variant: small-caps;
    • font – Shorthand for all font properties.
      Example: font: italic bold 16px/1.5 Arial, sans-serif;
  • Text Properties:
    • color – Sets the color of the text.
      Example: color: blue;
    • text-align – Aligns the text horizontally.
      Values: left, right, center, justify
      Example: text-align: center;
    • text-decoration – Adds decorative lines to text.
      Values: none, underline, overline, line-through
      Example: text-decoration: underline;
    • text-transform – Controls letter casing.
      Values: none, uppercase, lowercase, capitalize
      Example: text-transform: uppercase;
    • letter-spacing – Adjusts space between characters.
      Example: letter-spacing: 1.5px;
    • line-height – Sets space between lines.
      Example: line-height: 1.6;
    • word-spacing – Adjusts space between words.
      Example: word-spacing: 4px;
    • white-space – Controls handling of white spaces and line breaks.
      Values: normal, nowrap, pre, pre-line, pre-wrap
      Example: white-space: nowrap;
